# robofleet_server

*RoboFleet 2.0 Server*

## Dependencies

* Node.js
* [yarn](https://classic.yarnpkg.com/en/docs/install)

## Configuration

The server must be configured before building.
1. `cp src/config.example.ts src/config.ts`
2. Edit parameters in `src/config.ts`

## Building

* `yarn install`
* `yarn build` to build
* `yarn start` to start
